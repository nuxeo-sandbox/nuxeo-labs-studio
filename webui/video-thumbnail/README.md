# About

Module for setting the video poster and search thumbnail.

# Installation

1. Add `SetThumbnailByTimecode.js` as an automation script
1. Upload the UI elements to the Designer tab
1. Add the bundle.html snippet to your project's configuration in Designer

# Configuration

No other configuration is necessary to use this feature.

# Usage

Click on the 'burst-mode' icon for the Video to set the storyboard thumbnail.

# Support
